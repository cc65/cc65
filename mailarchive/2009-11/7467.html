I thought we were trying to come up with common color names, not
common color values...

If we want common color (or rather palette index) values across
targets then that means somewhat more remapping: The Apple2 doesn't
have a monochrome graphics mode so there would be no need to define
'white' as '1' in a target-specific include file. Therefore the 8
color Apple2 standard driver could go without remapping at all. I
don't know if there are other cases like this...

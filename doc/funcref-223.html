3.174 ser_status

Function

Return the serial port status.

Header

serial.h

Declaration

unsigned char __fastcall__ ser_status (unsigned char* status);

Description

Return the serial port status.

Limits

  • The function is only available as fastcall function, so it may only be used in presence of a prototype.

Availability
